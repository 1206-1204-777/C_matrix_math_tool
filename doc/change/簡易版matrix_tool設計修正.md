# 設計修正記録

## 1. 修正対象/修正概要
1. `create_matrix`,`add_matrix`: exit(1)を返す処理を`return NULL`に変更
2. `add_matrix`, `print_matrix`: 引数に`const`を付け値の変更を防ぐ
3. `create_matrix`: 構造体への代入時期の変更
4. `create_matrix`: callocの追加

## 2. 修正の内容
1. 対象関数にある`exit(1)`を全て`return NULL`に変更
2. `add_matrix`, `print_matrix`の引数全てに`const`を付ける
3. `m->rows = rows` と `m->cols = cols`の代入位置を`m->data = NULL`の直後に移動させる
4. `calloc`を導入することで0に初期化する処理を安全に実行できるようにする


## 3. 修正が必要と判断した理由
1. NULLを返すことで呼び出し元にエラー通知をしつつ、操作を続行させるか決めさせるため
2. `const`を付けることで呼び出し元で定義された値の変更を防ぐため
3. エラーが起きた際にどこまでの領域を確保しようとしたかについての情報を保持させるため
4. 現状では手動で確保した配列を初期化しているが、`calloc`の導入により自動で行い安全性を向上させるため

## 4. テスト範囲
- 全機能のテスト必須
